# 📖 About the Project

AI-Powered Virtual Study Room is a collaborative online learning platform that enables students to study together in real time. The platform combines live communication, AI-powered learning assistance, document sharing, and productivity tools to create an interactive and engaging virtual study environment.

---

# ✨ Features

### 👩‍🎓 Student Features

* Secure User Authentication
* Create & Join Study Rooms
* Real-Time Group Chat
* Document Upload & Sharing
* AI Study Assistant
* AI Notes Summarizer
* Automatic Question Generator
* Pomodoro Study Timer
* Room Management
* Responsive User Interface

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/yourusername/AI-Powered-Virtual-Study-Room.git
```

## Install Dependencies

```bash
npm install
```

## Start Backend

```bash
npm start
```

## Start Frontend

```bash
npm run dev
```
